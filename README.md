
# Star Wars Wiki Mobile App - Ionic & Angular

Mobile application developed with Ionic and Angular as part of a university practical project focused on learning hybrid mobile development concepts.

The application is inspired by a Star Wars Wiki and allows users to explore different categories such as people, planets, species and starships through a tab-based mobile interface. The project also includes a user login system, dynamic content rendering, reusable Angular components, routing, local storage management and responsive Ionic UI components.

## Features

* User authentication system
* Tab-based navigation
* Dynamic category and article visualization
* Star Wars themed content management
* Reusable Angular components
* Routing and page navigation
* Local storage integration
* Responsive mobile-oriented interface

## Technologies

* Ionic Framework
* Angular
* TypeScript
* HTML & SCSS
* Capacitor
* Ionic Storage

## Academic Purpose

Developed for learning the fundamentals of Ionic and Angular application development, including components, services, routing, storage handling and frontend interaction patterns.
